## ✅ 后端修正详情

### 任务 1: C→S 事件名修正

**文件**: [backend/app/sio.py](../backend/app/sio.py)

**改动**: 从 `@sio.event` 隐式注册改为 `@sio.on('event:name')` 显式注册

| 原注册方式 | 修正后 | 前端 emit |
|-----------|--------|----------|
| `@sio.event`<br>`async def create_table(...)` | `@sio.on('lobby:create_table')`<br>`async def create_table(...)` | `socket.emit('lobby:create_table', ...)` |

**修正的 9 个事件**:
- `lobby:list`
- `lobby:create_table`
- `lobby:join_table`
- `lobby:leave_table`
- `table:start_hand`
- `table:action`
- `table:add_bot`
- `table:remove_bot`
- `table:chat`

**验证**: `grep "@sio.on" backend/app/sio.py` 返回 9 个匹配 ✅

### 任务 2: table:hand_end 事件补充

**文件**: 
- [backend/app/sio.py](../backend/app/sio.py) (emit 逻辑)
- [backend/app/game/texas/engine.py](../backend/app/game/texas/engine.py) (Texas 结算)
- [backend/app/game/brag/engine.py](../backend/app/game/brag/engine.py) (Brag 结算)
- [backend/app/game/guandan/engine.py](../backend/app/game/guandan/engine.py) (Guandan 结算)

**新增方法**: 每个引擎实现 `get_hand_end_payload()` 返回契约定义的 payload:
```python
{
    "table_id": str,
    "hand_id": str,
    "results": [
        {
            "seat": int,
            "winnings": int,  # 正数=赢，负数=输
            "cards": [...],   # 摊牌时的手牌
            "rank": str       # 如 "Full House" (Texas), "豹子" (Brag)
        },
        ...
    ],
    "next_hand_in": 0  # 0 = 等手动 start_hand
}
```

**触发时机**: `sio.py` 的 `_broadcast_table_state()` 检测到 `hand_over=True` 时自动 emit。

**验证**: `grep "table:hand_end" backend/app/sio.py backend/app/game/*/engine.py` 返回 4+ 匹配 ✅
